Notes

  • WiFi and hostname changes need a reboot before they take effect.
  • Timezone, DST, calibration, and history settings are saved in NVS and can be adjusted without rebuilding the firmware.
  • Upload platforms can be disabled independently from this page if you only want local monitoring.
  • Credentials are stored in NVS on the device and are no longer limited to build-time secrets.
  • Tube presets live in logger_user_config.h so builders can ship cleaner defaults.
  • The compiled arduino_secrets.h values remain as fallback defaults.
  • The OTA update page is still served by the ElegantOTA library.
